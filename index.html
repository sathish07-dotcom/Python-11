Operator Overloading Python allows us to redefine the behavior of operators for user-defined classes. python Copy Edit class Point: def __init__(self, x, y): self.x = x self.y = y def __add__(self, other): return Point(self.x + other.x, self.y + other.y) def __str__(self): return f"({self.x}, {self.y})" p1 = Point(1, 2) p2 = Point(3, 4) print(p1 + p2) # (4, 6) 3.
